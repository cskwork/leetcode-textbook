## Summary

- **16 patterns**, **100 problems**.
- All Easy/Medium except three flagged capstones (LC 76, 212, and LC 5 which is officially
  Medium) -- each capstone is included because it is the canonical teaching instance of its
  pattern and skipping it leaves a gap.
- Two problems appear in two patterns (LC 347, LC 11) to demonstrate pattern overlap; each
  lives in a distinct folder with a distinct approach.
- Order within each pattern ramps Easy -> Medium.

This map drives the subagent generation. Each pattern = one subagent batch.
